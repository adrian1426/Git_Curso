--Apuntes de curso de Git

.-Ver ramas:
git branch

.-Crear una rama:
git checkout -b <nombre_rama>

.-Moverse a otra rama:
git checkout <nombre_rama>

.-Merge:
moverse a la rama destino y ejecutar el siguiente comando.
git merge <nombre_rama> (rama que se desea traer sus cambios)

.-Borrar ramas:
git branch -d <nombre_rama>

.-Borrar ramas Forzado:
git branch -D <nombre_rama>

.-Ver cambios disponibles en repositorio remoto
git fetch

.-Descargar o acatulizar el repositorio local
git pull

.-Ver tags
git tag

.-Tag, hacer una copia del repo y ya no puede ser modificado
git tag -a <nombre_tag> -m "mensaje tag"
